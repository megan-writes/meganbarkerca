## What this blog is for

Writing that doesn't belong anywhere else — thoughts on security, engineering, writing, or whatever else is worth saying.

## How to write a post

1. Create a new `.md` file in `src/content/blog/`
2. Fill in the frontmatter: `title`, `description`, `pubDate`, `tags`
3. Set `draft: false` when you're ready to publish
4. Push to `main` — Cloudflare Pages does the rest
