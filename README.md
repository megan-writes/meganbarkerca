# meganbarker.ca

Personal portfolio site for Megan Barker — security engineer and technical writer.

Built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com). Deployed on [Cloudflare Pages](https://pages.cloudflare.com).

## Local development

```sh
npm install
npm run dev
```

Runs at `localhost:4321`.

## Build

```sh
npm run build
```

Output goes to `dist/`. Deployment is automatic on push to `main` via Cloudflare Pages.
